## Why this blog?

Backend shuru karte time sab confuse ho jaate hain:

- kya seekhna hai?
- kaha se start kare?
- API kaise banate hain?

Is blog ka goal hai backend ko **simple aur practical tareeke se samjhana**.

---

## Introduction

Backend is the **brain of any application**.

Frontend dikhta hai,  
Backend sab kaam karta hai.

Backend handle karta hai:

- data storage
- authentication
- APIs
- server logic

---

## Rule 1: Basics clear kar

Frameworks se pehle basics samajh:

- HTTP methods (GET, POST, PUT, DELETE)
- Request & Response
- Status codes (200, 404, 500)
