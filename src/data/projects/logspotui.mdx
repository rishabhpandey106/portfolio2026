## Overview

**LogSpot UI** is a secure frontend package that provides an authenticated dashboard for viewing logs generated by LogSpot. It ensures that only authorized users can access log data through a protected interface.

## Key Features

- **Protected Dashboard**: Login-based access to logs  
- **Secure API Communication**  
- **Search & Filtering UI**  
- **Download Logs**  
- **Seamless Integration with LogSpot Backend**
